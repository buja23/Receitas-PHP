# Receitas em PHP

Um projeto simples em PHP para exibir uma lista de receitas a partir de um arquivo `JSON`.

## Como Rodar o Projeto

Existem duas maneiras principais de rodar este projeto:

### 1. Usando o Servidor Embutido do PHP (Recomendado)

Este é o método mais simples e rápido.

1.  Navegue até a pasta raiz do projeto (`Receitas-PHP`) pelo seu terminal.
2.  Execute o seguinte comando:

    ```bash
    php -S localhost:8000
    ```
3.  Abra seu navegador e acesse `http://localhost:8000`.

### 2.1 Usando um Servidor Web (XAMPP, WAMP, etc.)

Se você já tem um ambiente como XAMPP instalado (como o caminho do arquivo sugere):

1.  Certifique-se de que a pasta do projeto (`Receitas-PHP`) está dentro do diretório `htdocs` do seu XAMPP.
2.  Inicie o serviço Apache no painel de controle do XAMPP.
3.  Abra seu navegador e acesse `http://localhost/Receitas-PHP/`.

### 2.2 Usando um DOCKER

Se tiver docker instalado 

1. Abrir a pasta do projeto
2. rodar o seguinte comando:

    docker-compose up

 Abra seu navegador e acesse `http://localhost:8000`.
